Ajout d'un thème

Ajoutez un thème de séances de code ou le réactiver lorsqu'il a été virtuellement supprimé.

Les règles à respecter sont :
  • Le nom du thème est obligatoire.
  • Il ne peut pas contenir de caractères spéciaux, ni seulement d'espaces.
  • Si un thème portant le même nom existe déjà, alors :
    • Soit il est actif, auquel cas il n'est pas nécessaire d'en faire un doublon : un message est affiché et le doublon n'est pas créé.
    • Soit il a été virtuellement supprimé, dans ce cas il est réactivé et réapparaitra dans la liste des thèmes disponibles.
  • Le descriptif du thème est optionnel.
